# Notas : 
Aplicacion en consola de Node.JS que podemos utilizar para poder obtener en consola e imprimir resultados de las tablas.

```
  Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de la tabla de multiplicar                [número] [requerido]
  -l, --listar   Llamar opcion si desea que se imprima en consola
                                                                      [booleano] [defecto: false]
  -h, --hasta    Elige hasta que numero multiplicar                   [número] [defecto: 10]
  ```
